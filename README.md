# premium-calculator-python
A modern scientific calculator built using Python and Tkinter with advanced functions, memory system, history tracking, and premium UI design.

##  Features

### Basic Operations
- Addition (+)
- Subtraction (−)
- Multiplication (×)
- Division (÷)

### Scientific Functions
- sin, cos, tan
- log, ln
- √ (square root)
- x², x³
- π (pi)
- % (percentage)
- factorial (x!)

### Advanced Features
- Memory system (M+, M-, MR, MC)
- History tracking
- Save history to file
- Keyboard input support
- Backspace and clear (AC)
- Double zero (00) button
- Brackets support ( )

### UI/UX
- Premium dark theme
- Clean grid layout
- iPhone-inspired design
- Smooth and responsive interface

## Technologies Used

- Python
- Tkinter (GUI library)
